# ETK Model Project

This repository contains a multi-component AI system with:

- pulse_detector/: rPPG / remote photoplethysmography analysis module
- risk_model/: Qualcomm-style AI risk model training and inference pipeline
- frontend/: web UI for interacting with the system
- agent/: orchestration / automation layer

## Getting started

1. Review the README files in each subproject.
2. Install dependencies from the relevant requirements file.
3. Start with the risk model training/inference scripts.
